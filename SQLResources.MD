# SQL tutorials

You could start with any number of resources, but if you are just beginning to learn about databases, then learning about Structured Query Language is a good place.
Many database servers are one of these or something similar MSSQL, MySQL, PostgreSQL.  Yes, there are other Database types like Oracle, and there are other kinds of data storage like DynamoDB, GraphQL, NoSQL...but learning SQL is a common place to start.

This resource focuses on MSSQL.
